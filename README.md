｜目標：PDFを1ページずつバラして、それぞれOCRしてから一つの文字ファイルに纏める。
｜進捗：完成
｜実装:tesseract
｜機材：Linux
｜インプット：原稿PDF
｜アウトプット：原稿txtファイル
｜方法：linux OCR repository tesseract
｜ファイル：
